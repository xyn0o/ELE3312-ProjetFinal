void choosePlayer ( ili9341_t * lcd,
player_t * players )

The function draws the two possible players (Pac-Man or Ghost) on the screen. Additionally, it renders two hollow squares under each character and a filled square in the middle that serves as a cursor. The function is supposed to read the accelerometer value and use its values to move the cursor block until it coincides with one of the hollow squares. At this point, the player is chosen and the function returns.

Parameters
lcd[in] pointer to the ili9341 handle structure used to access the tft screen
players[out] array of player_t structures which contains the identities of the players when the function returns
